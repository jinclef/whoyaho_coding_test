
## 🛠 설치 및 실행 방법

```bash
git clone https://github.com/jinclef/whoyaho_coding_test.git
```

coding-test/ 에서 다음과 같이 실행해주세요.

```bash
npm install
npm run start
```

실행 후, 브라우저( http://localhost:8080/ )에서 **방향키(← ↑ → ↓)** 를 눌러 공을 움직여볼 수 있습니다.
